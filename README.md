## 🚀 Project Overview

In this project, I built a real-time Object Detection system using Python.  
The system uses a pre-trained YOLO model to detect and identify objects from live video input through a webcam.

This project helped me understand how computer vision works in real-world applications.

---

## ✨ Features

- 🎥 Real-time object detection using webcam
- 📦 Detects multiple objects simultaneously
- 🧠 Uses pre-trained YOLO model
- 🖼️ Displays bounding boxes and labels
- ⚡ Fast and efficient performance
- ✔ Simple and easy-to-understand implementation

---

## 🛠️ Technologies Used

- Python
- OpenCV (cv2)
- Ultralytics YOLO

---

## 📌 How It Works

1. The webcam captures live video frames  
2. Each frame is passed to the YOLO model  
3. The model detects objects present in the frame  
4. Bounding boxes and labels are drawn on detected objects  
5. The processed frame is displayed in real time
